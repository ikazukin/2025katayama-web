# カタヤマ企業 沿革データ

ACF（Advanced Custom Fields）の `timeline` リピーターフィールド用データ

## ACFフィールド構成

```
timeline (Repeater)
├─ year (Text) - 年
├─ month (Number) - 月（オプション）
├─ event_title (Text) - イベントタイトル
├─ event_description (Textarea) - イベント説明
├─ event_image (Image) - 単一画像（旧形式）
├─ event_gallery (Gallery) - 複数画像（カルーセル用）
├─ event_grid_gallery (Gallery) - グリッド表示用画像
└─ milestone (True/False) - 重要な出来事フラグ
```

## 📸 画像の追加方法

### 単一画像の場合
- `event_image` フィールドに1枚の画像をアップロード

### 複数画像（カルーセル）の場合
- `event_gallery` フィールドに複数枚をアップロード
- 自動的に横スクロールカルーセルで表示

### 複数画像（グリッド）の場合
- `event_grid_gallery` フィールドに複数枚をアップロード
- 3列グリッドで表示（Rebita風）

---

## 🎨 マイルストーンの使い方

重要な出来事には `milestone` を「はい」に設定すると：
- ⭐マークが表示される
- タイトルが青色（#004B9A）に強調される
- カードが目立つデザインになる

---

## ✅ ACF入力の推奨ワークフロー

1. **年ごとにまとめて入力**
   - 同じ年のイベントを連続で入力
   - 月順に並べる

2. **マイルストーンを厳選**
   - 創業、周年記念、認証取得など重要な出来事のみ
   - 全体の20%程度に抑える

3. **画像は厳選して追加**
   - すべてのイベントに画像は不要
   - 重要な出来事のみ追加

4. **説明文は簡潔に**
   - 1〜2行程度
   - 詳細は別ページで
