## 🔮 今後の実装予定

### 1. 見積もりシミュレーター機能

**参考サイト:** https://www.sunwall.co.jp/estimate/

**概要:**
ユーザーが簡単に大規模修繕工事の概算見積もりを確認できるインタラクティブなシミュレーター機能。

#### 想定される機能
- **建物情報の入力**
  - 建物の種類（マンション、アパート、ビル等）
  - 階数
  - 戸数
  - 築年数
  - 建物面積

- **工事内容の選択**
  - 外壁塗装
  - 防水工事
  - 鉄部塗装
  - タイル補修
  - シーリング工事
  - その他（オプション選択式）

- **概算見積もりの表示**
  - 工事項目ごとの単価計算
  - 合計金額の表示
  - 工期の目安表示
  - PDF出力機能（オプション）

- **お問い合わせへの導線**
  - シミュレーション結果をもとに、お問い合わせフォームへ自動入力
  - 見積もり依頼ボタン

#### 技術的な実装案
- **フロントエンド**
  - React.js または Vue.js でインタラクティブなUI構築
  - ステップ形式のフォーム（マルチステップフォーム）
  - リアルタイムで金額が更新される仕組み

- **バックエンド**
  - WordPress REST API 経由でデータ送信
  - Contact Form 7 または Flamingo と連携
  - 見積もりデータをデータベースに保存

- **データ管理**
  - ACFで単価設定を管理画面から編集可能に
  - 工事項目ごとの単価マスター
  - 建物タイプごとの係数設定

#### 実装優先度
🔴 **高**: ユーザーからの問い合わせ増加が見込める重要機能
